Storycard Dice Package

Use one package for server + browser API integration.

← Back to current demo

Setup (server.js → html)

// server.js
const { createDiceApi } = require('./packages/storycard-dice/serverApi');
const diceApi = createDiceApi();

// POST /api/dice
const die = diceApi.createDie({ sides: 6 });

// POST /api/dice/:id/roll
const roll = diceApi.rollDie(diceApi.getDie(die.id));

Live API demo

This card is created through the package API client and rolled through the same backend endpoints.

<!-- docs.html -->
<script type="module">
import { StorycardDiceApi, mountSingleDieDemo } from '/packages/storycard-dice/browser.js';

const api = new StorycardDiceApi('');
await mountSingleDieDemo({
  mount: document.getElementById('api-doc-demo'),
  api,
  sides: 6,
});
</script>

What this does

  • Creates a die from the front end through POST /api/dice.
  • Renders the die in a dedicated container card.
  • Rolls it with POST /api/dice/:id/roll when you press the button.
